Press A Key to Start

  1. Press any key on your keyboard to start.
  2. At each level, you much recreate the pattern the game reveals.
  3. The game will only reveal the newest addition to the pattern, but you have to recreate the pattern in its entirety including the newest addition.
  4. Once you have entered a pattern of the same length as the level you are on, the game will check if you are correct.
